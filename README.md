# GiftHub Insights

A modern e-commerce platform for personalized gifts.

## Features

- **Personalized Products**: Upload custom photos and add personalized text/dates.
- **Dynamic Shipping**: Location-based shipping calculations (Tamilnadu & Other States).
- **Responsive Design**: optimized for both desktop and mobile devices.
- **Premium UI**: Built with React, Tailwind CSS, and Framer Motion for smooth animations.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd gifthub-insights
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- **Frontend**: React, Vite, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **State Management**: Custom hook-based store
